Errai/GWT example A demo application demonstrating use of RichFaces standalone widgets to create a "polyglot" application using both RichFaces/JSF, Errai/GWT, and HTML/REST. The significant achievment here is a single application written in multiple languages/frameworks with a uniform L&F.
