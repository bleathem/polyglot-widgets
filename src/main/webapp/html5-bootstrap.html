The RichFaces standalone javascript widgets used with HTML5/Aerogear

Here we have a HTML5 Widget built using the standalone RichFaces Bootstrap "pickList" javascript widget, used to select/sort items from a list.

The pgae's javascript interacts directly with the pickList jQuery plugin: 

$('#myPickList').pickList();
